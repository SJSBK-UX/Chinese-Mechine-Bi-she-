# 源码提交说明

## 一、提交包内容

本提交包为《基于 YOLO11 的中药材检测与识别系统》源码整理版，已去除本机缓存、构建产物、日志文件、虚拟环境、IDE 配置、论文/PPT 中间产物和测试录屏文件。

提交包主要包含以下内容：

| 目录或文件 | 说明 |
| --- | --- |
| `MedicineDetection_vue` | Vue3 前端源码与配置文件 |
| `MedicineDetection_springboot` | Spring Boot 业务后端源码与 Maven 配置 |
| `MedicineDetection_flask` | Flask AI 推理后端源码、依赖说明和 YOLO 权重 |
| `MedicineDetection_mobile_app` | Android WebView 手机端源码与文档 |
| `yoloai.sql` | MySQL 数据库初始化脚本 |
| `start_all.bat` | Windows 一键启动入口 |
| `start_all.ps1` | PowerShell 一键启动脚本 |
| `项目说明书.md` | Markdown 版项目说明书 |
| `项目说明书.pdf` | PDF 版项目说明书 |
| `源码提交清单.md` | 源码提交清单 |

## 二、运行环境

建议运行环境如下：

| 环境 | 说明 |
| --- | --- |
| Python | Python 3.8 及以上 |
| Node.js | Node.js 16 及以上 |
| JDK | Java 8 或更高版本 |
| MySQL | MySQL 5.7 或 8.0 |

## 三、数据库说明

数据库脚本为：

```text
yoloai.sql
```

Spring Boot 默认数据库配置位于：

```text
MedicineDetection_springboot/src/main/resources/application.properties
```

默认配置为：

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/yolo?serverTimezone=Asia/Shanghai
spring.datasource.username=root
spring.datasource.password=123456
server.port=9999
```

如本机数据库用户名、密码或端口不同，请自行修改该配置。

## 四、启动方式

在项目根目录双击：

```text
start_all.bat
```

或在 PowerShell 中执行：

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\start_all.ps1
```

脚本会分别启动：

| 服务 | 默认端口 |
| --- | --- |
| Flask AI 推理服务 | 5000 |
| Spring Boot 后端 | 9999 |
| Vue 前端 | 8888 |

启动成功后访问：

```text
http://localhost:8888
```

## 六、模型文件说明

YOLO 模型文件位于：

```text
MedicineDetection_flask/weights/best.pt
MedicineDetection_flask/weights/best.onnx
```

其中 `best.pt` 用于 Python 环境下推理，`best.onnx` 可用于后续部署优化。

## 七、注意事项

DeepSeek 和 Qwen 的 API Key 未写入源码，Flask 后端通过环境变量读取：

```text
DEEPSEEK_API_KEY
QWEN_API_KEY
```

如果未配置 API Key，YOLO 图像检测功能仍可运行，但 AI 文字说明可能无法生成。
